盛最多水的容器

题目描述

这个题之前做过，相当于二刷了…

原有印象是双指针法，利用对撞指针求得面积最大的情况，与此同时，还能简化暴力求解的计算量。

盛水容器的最大值由如下式子描述 s = min(height[left],height[right])*(right-left)确定，在指针对撞过程中，应该是尽可能的发现最大的面积，（但是对撞移动指针的过程中，并不是说每一次移动都对应着最大的面积），我们需要在移动过程中，记录已被计算的面积的最大值。
