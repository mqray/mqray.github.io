leetcode_删除链表倒数第N个节点

发表于 2019-11-25 更新于 2019-11-28 分类于 leetcode 阅读次数： Valine：
本文字数： 1.3k 阅读时长 ≈ 1 分钟

给定一个链表，删除链表的倒数第 n 个节点，并且返回链表的头结点。

示例：

两次遍历

这种思路比较简单，第一次遍历得到单链表总长度，第二次遍历，有针对性的找到需要删除的节点，然后进行删除。由于可能出现的单链表为空，单独拿出来处理较为繁琐，所以在处理时，额外的增加一个头结点，然后再进行操作

实现代码

注意到，删除之后返回的是return p.next#为什么是删除p.next而不是head呢？因为如果只有一个元素，删除后head应该指向空，而如果返回head，它是指向原有节点的，虽然断开了连接，但是仍然指向初识单链表中的数

复杂度分析

移动次数总共是2*length-n，时间复杂度为2O(N)=O(N)

快慢指针

上面的解法中，我们通过求出单链表长度求解待删除数的下标进行删除，但是这个单链表长度本来是不需要求的？有没有优化方式呢？答案是有的。两次遍历的差异是，两次遍历指针总是相差输入n值。我们可以考虑使用两个指针，一个快一个慢，快的，先根据输入的n值，移动n次，然后两个指针一起向后移动，知道快的到达末尾。
