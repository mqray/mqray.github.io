如何使用模态对话框完成添加学生？？

首先要明白为什么要使用模态对话框？因为这里想实现异步请求和局部刷新，相应的会用到Ajax。

什么是Ajax？：Ajax是英文Asynchronous JavaScript And XML的缩写，指异步的JavaScript和XML，是一种创建快速动态的技术，通过后台与服务器进行少量的数据交互，实现页面的异步刷新，在不重新加载整个页面的情况下完成页面的部分刷新。

什么是异步、什么是同步：

  • 同步：发送方发出数据后，等接收方发回响应之后才能发下一个数据包的通讯方式。
  • 异步：发送方发出数据后，不等接收方发回响应，接着直接发送下一个数据包的通讯方式。

同步通讯和异步通讯的概念

  • 同步通信：要求通信双方已相同的时钟频率进行，准确协调，通过共享的一个单个时钟或定时脉冲源保证发送方和接收方的准确同步，效率教高。
  • 异步通信：异步通信方式不要求双方同步，收发双方可采用各自的时钟源，双方遵循一步的通信协议，以字符为数据传输单位，发送方传送字符的时间间隔不确定，发送效率比同步传送低。

通信选择：同步和异步

  • 同步：使用者通过使用单个线程调用服务：该线程发送请求，在服务运行时阻塞，并等待响应。
  • 异步：使用者通过两个线程调用服务：一个线程发送请求，另外一个单独线程接收响应。
