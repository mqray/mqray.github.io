最接近的三数之和

给定一个包括 n 个整数的数组 nums 和 一个目标值 target。找出 nums 中的三个整数，使得它们的和与 target 最接近。返回这三个数的和。假定每组输入只存在唯一答案。

例如，给定数组 nums = [-1，2，1，-4], 和 target = 1.

与 target 最接近的三个数的和为 2. (-1 + 2 + 1 = 2).

双指针对撞

和上一题的思路很像，差别在于之前的target为0，而现在的target需要指定。那么在判别时应注意哪些问题呢？

这里沿用之前的表达，用limit = nums[i]+nums[left]+nums[right]迭代和对撞指针过程中产生组合数的和。

  1. 首先，这个target与limit之间的差值是变动的，它可能会有一个从正到负的变化，也可能会有从负到正的变化，而我们只关注差值的最小值
  2. 在双指针对撞的过程中，我们应该关注每一次对撞得到的limit的值与target差值的绝对值，并实时更新这个差值【取绝对值最小的差值】
  3. 同时，对于产生的绝对值最小的差值，我们应当记录下此刻的limit的结果，因为题目要求返回的是这三个数的和，而不是对应的数字组合。
