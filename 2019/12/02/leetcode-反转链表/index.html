反转链表II

反转从位置 m 到 n 的链表。请使用一趟扫描完成反转。

说明:
1 ≤ m ≤ n ≤ 链表长度。

迭代思路

这里还是沿用之前迭代的思路，对于要反转的部分进行反转即可，由于反转列表前后可能存在其他节点，所以再反转过程中应注意一下几点：

  • 首先应该使用指针定位到需要反转操作的部分，用前后两个指针确定反转范围，要记录反转链表左侧的第一个节点flag1，便于连接反转之后的链表。
  • 对于反转部分使用迭代/递归均可完成
  • 对于完成反转之后的部分应该与原始链表右边部分相连，所以，反转链表的初始头结点需要被记录，相应的原始链表右侧的节点也应该记录，即flag2
  • 最后利用first.next = second和flag1.next = flag2连接整个链表即可。

其时间复杂度为O(N)，空间复杂度为O(1)
